# Baseball Project

R analysis exploring relationships between MLB pitching metrics and WAR from 2021 through 2025.

## Contents

- `Sports project.R` - main R analysis script.
- `WAR2021.csv` through `WAR2025.csv` - WAR data by season.
- `active-spin21.csv` through `active-spin25.csv` - active spin data by season.
- `pitch_movement21.csv` through `pitch_movement25.csv` - pitch movement data by season.
- `Acuna Stats + Sprint Speed.csv` and `.xlsx` - Ronald Acuna Jr. stats and sprint speed data.
- `Rplot*.pdf` and screenshots - generated plots/output artifacts.

## Requirements

The R script uses:

- `dplyr`
- `tidyr`
- `ggplot2`

## Run

Open `Sports project.R` in RStudio or run it from this folder in R.
